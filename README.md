OpenIE - A simple utility that launches Internet Explorer on Windows 10+
<br>
Windows 10 이상에서 Internet Explorer를 실행하는 프로그램입니다.

# Usage
The default start page is set to Google, but you can specify a custom URL by adding a parameter to the shortcut.
<br>
기본 시작 페이지는 Google로 설정되어 있으며, 바로 가기에 매개 변수를 추가하면 원하는 URL로 지정할 수 있습니다.

ex)
<br>
OpenIE.exe https://www.naver.com
<br>
OpenIE.exe www.naver.com
